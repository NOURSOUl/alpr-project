"# ALPR Project - Automatic License Plate Recognition" 
"" 
"## Project Overview" 
"A complete license plate recognition system built with Python, OpenCV, and EasyOCR." 
"" 
"## Features" 
"- Multiple plate detection methods" 
"- Graphical User Interface (GUI)" 
"- OpenCV cascade classifier" 
"- EasyOCR integration" 
"- Test images included" 
"" 
"## Usage" 
"1. **GUI Version:** Run \`python alpr/alpr_final_gui.py\`" 
"2. **Command Line Version:** Run \`python alpr/alpr_final.py\`" 
"3. **Basic Version:** Run \`python alpr/license_plate_recognition.py\`" 
"" 
"## Requirements" 
"Install dependencies: \`pip install opencv-python easyocr numpy Pillow\`"
